# Origin Studio

> **Slogan:** Sites modernos que colocam seu negócio no mapa.

## Marca

- **Nome:** Origin Studio  
- **Símbolo:** ponto de origem + órbitas (expansão digital)

## Stack

Next.js · TypeScript · Tailwind CSS

```bash
npm install
npm run dev
```
